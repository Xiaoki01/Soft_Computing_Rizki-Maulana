Tugas 1: Pengenalan Soft Computing

1. Pengertian Soft Computing

Soft Computing adalah pendekatan komputasi yang digunakan untuk menyelesaikan masalah kompleks yang sulit dimodelkan secara matematis. Soft Computing lebih fleksibel, toleran terhadap ketidakpastian, dan mampu mendekati solusi optimal meskipun data tidak lengkap.

Karakteristik Soft Computing:
  • Toleran terhadap ketidakpastian dan ketidakakuratan
  • Fleksibel dan adaptif terhadap perubahan data
  • Mampu memberikan solusi mendekati optimal
  • Menggabungkan berbagai teknik kecerdasan buatan
Komponen Utama:
  • Fuzzy Logic (Logika Fuzzy)
  • Neural Network (Jaringan Saraf Tiruan)
  • Genetic Algorithm (Algoritma Genetika)
  • Machine Learning
2. Logika Fuzzy (Fuzzy Logic)

Logika Fuzzy memperluas konsep logika biner (benar/salah) menjadi spektrum nilai kebenaran antara 0 dan 1. Hal ini meniru cara manusia berpikir dengan ketidakpastian, seperti "agak panas", "cukup cepat", dan sebagainya. Pendekatan ini digunakan ketika model matematis sulit atau tidak memungkinkan dibuat, namun keputusan tetap harus dihasilkan secara realistis seperti pemikiran manusia.

Konsep Dasar:
  • Himpunan Fuzzy: Elemen memiliki derajat keanggotaan (0–1)
  • Fungsi Keanggotaan: Menentukan sejauh mana nilai masuk dalam suatu kategori
  • Aturan Fuzzy: IF–THEN logika linguistik untuk pengambilan keputusan
  • Defuzzifikasi: Konversi hasil fuzzy menjadi nilai tegas (crisp)
Sejarah Singkat

Konsep himpunan fuzzy diperkenalkan oleh Lotfi A. Zadeh pada tahun 1965 (University of California, Berkeley). Penerapan awal untuk kontrol linguistik dikembangkan oleh Ebrahim H. Mamdani dan S. Assilian pada 1975 (uji coba pengendalian boiler/steam engine menggunakan aturan linguistik fuzzy). Tokoh lain yang berkontribusi pada variasi metode fuzzy termasuk Tsukamoto dan para peneliti bidang informasi/pencarian yang mengadaptasi konsep fuzzy untuk tolerant matching.

Perkembangan

Logika fuzzy saat ini digunakan secara luas pada sistem kontrol cerdas, seperti AC otomatis , transmisi otomatis mobil, kamera autofokus, hingga sistem pendukung keputusan medis. Teknologi ini terus berkembang dan digabungkan dengan neural networks menjadi “Neuro-Fuzzy” untuk meningkatkan akurasi adaptasi.

Metode: Mamdani

Mamdani menggunakan aturan IF–THEN berbasis himpunan fuzzy untuk input dan output. Inferensi menghasilkan himpunan fuzzy, kemudian dilakukan proses defuzzification (umumnya menggunakan metode centroid) untuk menghasilkan nilai crisp. Metode ini lebih intuitif karena menyerupai cara berpikir manusia.

Alur Operasi: Fuzzification → Evaluasi Aturan (min/max atau t-norm/t-conorm) → Agregasi → Defuzzification (centroid).

Cocok untuk sistem berbasis kontrol yang butuh interpretasi bahasa natural, misalnya sistem pendingin ruangan, kecepatan kipas, atau kontrol lalu lintas. Digunakan saat ketepatan numerik tidak terlalu ketat, tetapi transparansi aturan tinggi. Representatif dan mudah dipahami pakar, cocok untuk aplikasi yang butuh interpretabilitas tinggi.

Metode: Sugeno

Sugeno menggunakan aturan IF–THEN, tetapi bagian konsekuen (output) berupa fungsi linear atau konstanta, bukan himpunan fuzzy. Hasil inferensi langsung berupa nilai numerik sehingga tidak memerlukan proses defuzzifikasi berat seperti Mamdani.

Alur Operasi: Fuzzification → Evaluasi Aturan → Pembobotan Output Linear → Pembagian Tertimbang (Weighted Average).

Cocok pada sistem yang membutuhkan perhitungan cepat dan presisi numerik lebih tinggi, misalnya prediksi, optimasi, sistem adaptif, dan model kontrol otomatis modern. Perhitungan lebih ringan dan mudah dipadukan dengan algoritma optimasi (mis. ANN, ANFIS), cocok untuk data real-time.

Metode: Tsukamoto

Pada metode Tsukamoto, setiap aturan IF–THEN menghasilkan output crisp secara langsung melalui fungsi keanggotaan monoton. Output akhir diperoleh dari rata-rata tertimbang berdasarkan derajat keanggotaan aturan.

Alur Operasi: Fuzzification → Evaluasi Aturan → Perhitungan Output Crisp Tiap Aturan → Rata-Rata Tertimbang.

Cocok untuk proses yang membutuhkan hubungan input-output yang jelas dan terkontrol secara numerik, serta saat urutan nilai harus terjaga (fungsi monoton), seperti sistem penilaian risiko atau kredit. Output setiap aturan langsung berupa nilai tegas sehingga lebih stabil untuk keputusan yang sensitif terhadap perubahan kecil pada input.

Contoh Rumus Fungsi Keanggotaan Segitiga:

μ(x) = 0,                
jika x ≤ a atau x ≥ c
(x - a)/(b - a), jika a < x ≤ b
(c - x)/(c - b), jika b < x < c

Di mana μ(x) menunjukkan derajat keanggotaan nilai x terhadap himpunan fuzzy dengan titik a, b, dan c.

Contoh Aplikasi:
  • IF suhu = "hangat" THEN kipas = "sedang"
  • IF suhu = "panas" THEN kipas = "cepat"

Logika fuzzy memungkinkan sistem menentukan kecepatan kipas berdasarkan suhu aktual (misalnya 28°C memiliki "hangat" = 0.6 dan "panas" = 0.4).

3. Neural Network (Jaringan Saraf Tiruan)

Neural Network atau jaringan saraf tiruan adalah sistem komputasi yang meniru cara kerja otak manusia.Terdiri dari neuron-neuron buatan yang saling terhubung dan mampu belajar dari data. Penting karena kemampuan generalisasi dari data berisik, ekstraksi fitur otomatis, dan fleksibilitas untuk berbagai tugas seperti klasifikasi, regresi, pemrosesan citra, dan pemahaman bahasa alami.

Komponen:
  • Input Layer: menerima data
  • Hidden Layer: memproses informasi
  • Output Layer: menghasilkan output
  • Weights: mengatur kekuatan koneksi antar neuron
  • Activation Function: menentukan respons neuron
Sejarah Singkat

  • 1943: Warren McCulloch & Walter Pitts — model neuron biner formal pertama.
  • 1958: Frank Rosenblatt — Perceptron, model pembelajaran linear awal untuk pengenalan pola.
  • 1986: David E. Rumelhart, Geoffrey Hinton & Ronald J. Williams — popularisasi algoritma backpropagation untuk melatih jaringan multi-layer (MLP), yang membuka jalan untuk jaringan yang lebih dalam.
  • 1990-an — 2010-an: Kontribusi signifikan dari Yann LeCun (CNN), Sepp Hochreiter & Jürgen Schmidhuber (LSTM), serta riset yang mengarah pada deep learning modern oleh Hinton, LeCun, dan Bengio.

Rumus Dasar Aktivasi Neuron:

z = Σ(wᵢ * xᵢ) + b
f(z) = 1 / (1 + e^(-z))      ← Fungsi aktivasi sigmoid
Proses Belajar (Backpropagation):

Δw = -η * (∂E/∂w)

Di mana:

  • Δw = perubahan bobot
  • η = learning rate
  • E = error total
Proses ini dilakukan berulang hingga error minimum tercapai.

Contoh Aplikasi:
  • Pengenalan gambar (CNN)
  • Penerjemahan bahasa (RNN, LSTM)
  • Prediksi cuaca dan harga saham
  • Diagnosa medis otomatis
